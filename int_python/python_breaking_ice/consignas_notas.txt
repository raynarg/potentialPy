Machete:
    En python hay tipos de datos primitivos y colecciones
    Diferencia mutable e inmutable, ejemplo con int:
        Si hacemos
            x = 5
            x = x + 1
            No estamos modificando <x> sino que estamos haciendo que <x> apunte a otro objeto nuevo (x+1=6)
            PERO! 5 sigue existiendo (NO MUTA)

    Primitivos:
        numeros (float, int, complex)
        strings (str)
        booleanos (bool)
    Colecciones
        Mutables
            listas (list)
            diccionarios (dict) / clave:valor
            conjuntos (set) / sin duplicados
        No-Mutables
            tuplas (tuple)
Para verificare el tipo de dato de una variable utilzamos type()
    - ej. Devuelve si pertenece a la clase 'int''float', etc.

        

🔁 Ejercicio 1: Calculadora simple
Pedí dos números y una operación (+, -, *, /) al usuario.
Mostrá el resultado según la operación elegida.

Validá que no se divida por cero.

📆 Ejercicio 2: Año bisiesto
Pedí un año al usuario y determiná si es bisiesto.

Es bisiesto si:
(año % 4 == 0 and año % 100 != 0) or (año % 400 == 0)

🔠 Ejercicio 3: Mayúscula o minúscula
Pedí una letra e indicá si es mayúscula, minúscula, o si no es una letra.

🎰 Ejercicio 4: Última cifra
Pedí un número entero y mostrá su última cifra (por ejemplo, de 123, debe mostrar 3).

📊 Ejercicio 5: Clasificación de edades

0-12: Niño

13-17: Adolescente

18-64: Adulto

🔄 Ejercicio 6: Número invertido (3 cifras)
Pedí un número de 3 cifras y mostrálo al revés.

Por ejemplo, si el usuario escribe 321, el programa debe imprimir 123.

🧮 Ejercicio 7: Evaluador de expresión

🎤 Ejercicio 8: Detectar vocales en texto

🔡 Ejercicio 9: Palabra al revés

🎯 Ejercicio 10: Últimos dos dígitos del DNI
